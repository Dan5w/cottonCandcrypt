# cottonCandcrypt

Para la instalacion es necesario correr [setup.bat], 
este instalara las depencias necesarios especificadas en [requirements.txt]
luego pedira las credenciales de tu BD de MySQL para inicializar la script [init_db.py]
que se encargara de crear la bd necesaria para la creacion del sistema.

Para correr el sistema se inicaliza con el comando >>python app.py
